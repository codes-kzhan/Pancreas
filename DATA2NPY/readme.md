# Pancreas

- dataset

https://wiki.cancerimagingarchive.net/display/Public/Pancreas-CT

- Install dicom

see also: https://pydicom.readthedocs.io/en/latest/getting_started.html#installing
```
$pip install pydicom
```
- Install nibabel
```
$pip install nibabel
```

- Convert Data to .npy
```
$python dicom2npy.py
$python nii2npy.py
```
